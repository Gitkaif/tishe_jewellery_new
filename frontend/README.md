# Customs4U - Gaming Tournament Platform

A modern web application for managing and participating in gaming tournaments and custom rooms, built with React and Tailwind CSS.

## Features

- Browse and join gaming tournaments for BGMI, Free Fire, and Valorant
- Participate in free and paid custom rooms
- User authentication system
- Responsive design with gaming theme
- Real-time tournament updates

## Tech Stack

- React
- Tailwind CSS
- React Router
- Vite

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Project Structure

- `/src/components` - Reusable UI components
- `/src/pages` - Page components for different routes
- `/src/assets` - Static assets like images
- `/public` - Public assets and index.html
